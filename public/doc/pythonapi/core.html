    Camera

    class Camera(view: ~brayns.utils.view.view.View = <factory>, projection: ~brayns.core.camera.projection.Projection = <factory>)

    Bases: object

    Camera used to render.

    A camera is composed of a View and a Projection.

    The view defines the camera position and orientation while the projection defines how the model will be projected from 3D to 2D.

    To focus the camera to a given target (bounds) use look_at to set the default full-screen front view.

    The front view has X-right, Y-up and Z-forward.

    Then, to rotate the camera around its target, use rotate_around_target.
